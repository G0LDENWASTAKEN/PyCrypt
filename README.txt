HOW TO INSTALL IF YOU DON'T HAVE PYTHON:


Step 1: Run "compile.bat".
Step 2: Wait for the command prompt window to close.
Step 3: Open the folder named "dist"


The file should be in that folder.


Common errors:


(When opening "compile.bat")
"pyinstaller" is not recognized as an internal or external command,
operable program or batch file.

You don't have pyinstaller on your PC, go to cmd, and type "pip install pyinstaller"


"pip" is not recognized as an internal or external command,
operable program or batch file.

You don't have pip installed, or it isn't added to your Path, open this link for information on how to fix: https://bit.ly/3FWOxLn


"dist" folder doesn't show up!

Right click in a blank part of the directory, and press "refresh"

If you're on Windows 11, Do the same, except press "show more options", then press "refresh"
